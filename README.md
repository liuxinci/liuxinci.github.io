# 个人主页介绍 — 刘新慈

## 项目概述

城市地下空间工程专业学生刘新慈的个人介绍主页，参考 [qiang-zou.github.io](https://qiang-zou.github.io/) 的学术个人主页风格，采用单文件 HTML 实现，最终部署到 GitHub Pages。

## 方案选择

**方案 A：单文件 HTML（纯前端）**

- 一个 `index.html` 文件，无构建步骤
- 零依赖，浏览器直接打开即可预览
- 推送到 GitHub Pages 即可部署

## 配色方案（肉色系暖调）

| 用途 | 颜色代码 | 说明 |
|------|----------|------|
| 背景色 | `#FDF8F4` | 极浅米白 |
| 卡片/区域背景 | `#FAF0E6` | 亚麻色 |
| 主标题/品牌色 | `#D4A574` | 暖沙色 |

## 内容板块

1. **关于我** — 简短自我介绍
2. **教育经历** — 学校 · 专业 · 时间
3. **项目经历** — 卡片式展示，带描述
4. **个人技能** — 标签式展示
5. **相关荣誉** — 奖项/证书列表
6. **视频作品** — MP4 视频网格展示，点击播放

## 技术要点

- ✅ 单文件 `index.html`，零依赖
- ✅ 响应式布局（手机/电脑自适应）
- ✅ MP4 视频 `<video>` 标签 + 自定义播放按钮
- ✅ 平滑滚动 + 渐入动画
- ✅ 侧边栏固定（大屏）/ 折叠（小屏）
- ✅ 肉色调暖系视觉风格

## 部署方式

将 `index.html` 及 `media/`（视频文件）推送到 GitHub Pages 仓库即可。
